Background and state in specs

'transform' CSS property enables transformation of box, like scaling one axis of character. This property is applicable to transformable elements, which is defined as all elements in CSS box model except for non-replaced and inline boxes etc.

'ruby' elements has 'ruby' inner display layout and 'ryby-base' and 'ruby-text' have 'flow' inner display type.

Therefore, 'transform' will not be applicable to 'rt' element per current spec.

Alternatively, font-stretch could be used.
